## ESI PROTOCOL CONSIDERATIONS

For electronic discovery, consider requesting:

1. **Native Format Production**
   - Request documents in native format with metadata
   - Specify acceptable formats (PDF, TIFF, native)

2. **Search Terms**
   - Agree on search terms for ESI
   - Document search methodology

3. **Custodians**
   - Identify key custodians
   - Request preservation from specific individuals

4. **Date Ranges**
   - Specify relevant time periods
   - Consider pre-litigation communications

## PRIVILEGE LOG REQUIREMENTS

If documents are withheld, the log must include:

| Field | Description |
|-------|-------------|
| Date | Date of document |
| Author | Who created it |
| Recipients | Who received it |
| CC | Carbon copy recipients |
| Subject | Brief description |
| Privilege | Attorney-client, work product, etc. |
| Basis | Why privilege applies |
