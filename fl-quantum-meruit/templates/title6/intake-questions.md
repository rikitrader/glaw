# C. CAUSE-SPECIFIC INTAKE (ask only for the routed claim)
- **Replevin:** exact description + serial #s + value of each item; where located; how detention began.
- **Ejectment/Quiet title:** legal description; source of title (deed book/page); the adverse instrument.
- **Eviction:** lease; rent amount + months due; which notice was served + when + how; registry deposit issue.
- **Partition:** all cotenants + shares; mortgages/liens of record; in-kind feasible or sale needed.
- **Declaratory:** the instrument/statute; the precise question; the present controversy.
- **Injunction:** the conduct to enjoin; the irreparable harm; can a bond be posted.
- **Lost instrument:** the document's terms; how/when lost; search efforts.
- **Dishonored check:** check copy; the § 68.065 demand letter sent? date?
- **Statutory lien:** lien type; perfection facts (NTO, claim-of-lien dates for Ch. 713); 1-yr clock.
- **Attachment/Garnishment:** the asset/garnishee; bond capacity; head-of-family status of debtor.

# D. PRE-FILE GATES (confirm before drafting)
1. Limitations period not run (verify per claim). 2. All conditions precedent / statutory demands &
notices satisfied. 3. Correct court + venue (Ch. 47). 4. Proper service path (Ch. 48/49; long-arm).
5. Cites verified (/glaw-legal-research) + RED→BLUE (/glaw-adversarial). 6. UPL/work-product footer.
