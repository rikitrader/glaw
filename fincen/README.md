# GLAW FinCEN Cell — Self-Contained Knowledge Base

The FinCEN cell directs financial-crime intelligence and BSA/AML regulatory-change tracking
for a GLAW matter. This umbrella seat (`/glaw-fincen`) and its five sub-seats are
**self-contained**: each carries its own `references/` knowledge base grounded in the user's
subscribed **FinCEN Updates** corpus (2025–2026) plus the standing BSA/AML framework.

## Seats
| Seat | Skill | Scope |
|---|---|---|
| Umbrella (CFIO) | `/glaw-fincen` | Directs the cell; standing BSA/AML KB + regulatory-change ledger |
| AML | `/glaw-fincen-aml` | Full AML investigation: placement/layering/integration, beneficial ownership |
| SAR | `/glaw-fincen-sar` | Suspicious-activity typologies & red-flag analysis (not a filed SAR) |
| Crypto | `/glaw-fincen-crypto` | On-chain tracing; GENIUS Act / stablecoin BSA frontier |
| OFAC | `/glaw-fincen-ofac` | Sanctions screening, 50% rule, evasion; FATF/IRGC interplay |
| TBML | `/glaw-fincen-tbml` | Trade-based ML; GTOs, border/cartel typologies |

## Umbrella reference files (`references/`)
- `regulatory-updates-2025-2026.md` — **the core**: chronological ledger of every FinCEN
  Update (2025–2026), grouped A–I, each with type, substance, authority, and compliance impact.
- `bsa-aml-framework.md` — standing framework: 31 U.S.C. 5311 et seq., 31 CFR Ch. X, the five
  pillars, the AMLA-2020 shift, BSAAG, examination.
- `cdd-beneficial-ownership.md` — CDD Rule four prongs; 2026 exceptive relief + consolidated
  FAQs; CTA/BOI reporting status (volatile).
- `sar-ctr-reporting.md` — SAR/CTR thresholds & timing; Oct-2025 SAR FAQs; 314(a)/(b).
- `genius-act-stablecoins.md` — GENIUS Act, PPSIs as BSA FIs, the CIP NPRM, crypto/MSB duties.
- `gto-tracker.md` — active GTOs (Southwest Border, Minnesota), thresholds, exemptive relief.
- `enforcement-actions.md` — Canaccord $80M, Paxful $3.5M, Brink's, Asre — facts + lessons.
- `fatf-international.md` — FATF lists (Nov 2025), cross-border sharing, sanctions interplay.
- `whistleblower-program.md` — AML/sanctions whistleblower program (AMLA §6314 / 31 USC 5323).
- `persona-and-guardrails.md` — tone, UPL/"not a filing", zero-fabrication, proposed≠final.
- `sources-corpus-index.md` — provenance: each email → KB file + primary-authority map.

Each sub-seat carries `references/regulatory-updates.md` — its slice of the ledger,
cross-referencing this umbrella's `regulatory-updates-2025-2026.md`.

## Posture
Analytical/compliance-advisory work-product for a licensed professional. **Not** a SAR/CTR
filing, an OFAC determination, or a charging/licensing decision. Many 2025–2026 items are
**proposed rules** — verify current status on FinCEN.gov / federalregister.gov before relying.
