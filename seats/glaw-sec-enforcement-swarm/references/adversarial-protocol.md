# Adversarial Protocol — RED → BLUE

No conclusion ships un-attacked. After the substantive seats produce findings and they are
reconciled, run this pass. Spawn parallel adversarial sub-agents (Agent tool, one RED per
claim, one BLUE per survivor) or run it inline for small matters.

## RED TEAM — destroy each claim

The RED agent's posture flips with the engagement posture:

- **Investigative matter** → RED is **defense counsel for the subject.** Mission: kill the
  charge before it's filed.
- **Defense matter** → RED is **SEC Enforcement staff.** Mission: tear apart the planned
  Wells submission / defense before staff does.

### Standard attacks, per cause of action

**10b-5 / §17(a) fraud**
- Materiality: would a reasonable investor really care? Puffery? Already public ("truth on
  the market")? Immaterial quantitatively & qualitatively (SAB 99)?
- Scienter: where is intent or recklessness? Good-faith reliance on auditors/counsel? An
  honest mistake, not fraud? (For §17(a)(2)/(3), note negligence suffices — don't over-claim.)
- Falsity: was the statement actually false *when made*? Opinion vs. fact (*Omnicare*)?
- "In connection with": is there a real purchase/sale nexus?

**Insider trading**
- Was the information actually **material AND nonpublic**?
- Duty: what duty was breached, owed to whom? (classical vs. misappropriation)
- Tipping: where is the **personal benefit** to the tipper (*Dirks/Salman*; *Newman*)?
- Causation/innocent explanation: pre-existing 10b5-1 plan? Independent research? Coincidental
  timing? Routine, pre-scheduled trade?

**Market manipulation**
- Were the trades **non-bona-fide**? Legitimate hedging/market-making explanation?
- Intent to create artificial price vs. ordinary aggressive trading?
- Does the tape actually show the pattern, or is it pattern-matching on noise?

**FCPA**
- Was the recipient a **"foreign official"**? Was there **corrupt intent** vs. a facilitating
  payment / bona-fide promotional expense / lawful-under-local-law defense?
- Knowledge of the third-party intermediary's conduct, or mere negligence?

**Remedies / procedure (always)**
- **Statute of limitations** (28 U.S.C. §2462 5-yr penalties; *Gabelli/Kokesh*; NDAA 10-yr
  scienter disgorgement). Is any of it time-barred?
- ***Liu*** — is the disgorgement figure net of legitimate expenses and tied to victim return?
- Jurisdiction/extraterritoriality (*Morrison*). Standing. Reliance/loss causation (if private).

## BLUE TEAM — rebuild only the survivors

For each claim RED attacked:
1. **Did it survive?** If a core element is genuinely unprovable on the record → **DROP** it
   (say so plainly) or demote to a *lead* needing more evidence.
2. **If it survives**, state the **strengthened theory**: the exact evidence (with cites)
   that defeats each specific RED attack.
3. **Residual risk**: note the best surviving defense even for kept claims — never present a
   claim as bulletproof.

## Output of this pass

- **RED-Team Kill Report** — per claim: every attack + whether it lands.
- **BLUE-Team Surviving Case** — kept claims, strengthened theory, evidence that beats RED,
  residual risk, and a **confidence tag** (High/Medium/Low + why).
- Dropped/demoted claims listed explicitly with the reason.

## Exhaustive mode (litigation-grade)
- Run **two independent RED teams** per claim and keep a claim only if it survives both.
- Add a **completeness critic**: "What element, document, witness, or alternative theory did
  we fail to test?" Its output seeds the Gaps & Next Steps section.
