# tax-relief

Tax-debt **resolution** specialist for people who **can't pay the IRS** — finds the legitimate path
and explains the **IRS Fresh Start** initiative honestly (the antidote to "pennies on the dollar"
OIC mills). The "can't-pay resolution" front door of the tax workflow.

## What it does
Triages active collection (stop the levy), runs the **compliance gate** (must file first → routes to
`tax-compliance`), **pre-qualifies** the taxpayer with the RCP math, and picks the real path:
short-term plan / **streamlined IA ($50k/72mo)** / **PPIA** / **Offer in Compromise** / **Currently
Not Collectible** / penalty abatement / innocent spouse — then builds and files it.

## Files
- `SKILL.md` — 6-step workflow with the compliance gate and honest reality framing.
- `references/persona-and-reality.md` — **Read first.** Fresh-Start-≠-forgiveness, anti-OIC-mill stance, ethics, escalation.
- `references/fresh-start-programs.md` — Streamlined IA, lien rules ($10k threshold + withdrawal w/ DDIA), expanded OIC ($205 fee/20% down/low-income waiver), PPIA, penalty relief.
- `references/choose-the-path.md` — Decision tree + **RCP estimator** to pre-qualify OIC vs PPIA vs CNC vs IA.
- `references/lien-levy-and-hardship.md` — Stop/release levies & garnishments, lien withdrawal/discharge/subordination, CDP (12153), TAS (911), CNC, CSED.
- `references/representation-and-scams.md` — DIY vs EA/CPA/attorney, how OIC mills work + IRS warnings, red flags, hire checklist.
- `references/worked-example.md` — $38k owed, one unfiled year, wage-levy notice — gate → triage → pre-qualify → resolve.

## Workflow ties
- **`tax-compliance`** — prerequisite (file all returns first); shared references reused, not duplicated (collection-resolution, penalty-relief, statutes-and-rights, letter-templates).
- **`tax-strategy`** — plan forward after the debt is resolved.
- **`financial-forensics`** — build the 433 financial statement.

## Scope & limits
Informational, **not a guarantee** of OIC acceptance, **not a substitute** for a licensed CPA/EA/
tax attorney. Refuses scams and false financial statements. Current figures verified live (June
2026): Fresh Start streamlined IA $50k/72mo + two tiers; lien threshold ~$10k; OIC $205 fee, 250%-
poverty low-income waiver, 20% lump-sum down, RCP method; CNC pauses (not forgiveness). Verify on
IRS.gov.
