# Formation Documents & Sequence

Verify current state forms/fees per SKILL.md Step 1a. Order matters — file the charter first, then
organize, then issue equity.

## 1. The charter (state filing that creates the entity)
- **Corporation:** *Certificate of Incorporation* (DE) / *Articles of Incorporation* (most states).
  Required: name (+ availability check/reservation), registered agent + office, **authorized
  shares** (number + par + classes — set up Class A/B here if dual-class, see founder-control),
  incorporator, purpose, optional §102(b)(7) director-exculpation, indemnification.
- **LLC:** *Articles/Certificate of Organization*. Minimal — name, registered agent; the
  substance lives in the **operating agreement** (Step 4 / governance).
- **PBC:** certificate states the **specific public benefit**. **Nonprofit:** articles with §501(c)(3)
  purpose + dissolution clause (for later IRS Form 1023).

## 2. Registered agent + registered office
Every entity needs one in its state of formation **and** in each state where it foreign-qualifies.
Use a commercial agent (CT, CSC, Registered Agents Inc.) if no in-state address.

## 3. EIN — IRS Form SS-4
Federal tax ID; needed to open a bank account, hire, and file. Foreign owners without an SSN/ITIN
apply by fax/mail (responsible-party rules).

## 4. Organizational consent / initial minutes
The first governance act (`templates/organizational-consent.md`):
- Adopt **bylaws** (corp) / acknowledge **operating agreement** (LLC).
- Appoint initial directors & officers (corp) or managers (LLC).
- Authorize issuance of founder stock/units; approve the cap table.
- Banking resolution; fiscal year; authorize the S-election if applicable.

## 5. Tax elections
- **S-election — Form 2553:** file within **2 months 15 days** of the start of the tax year it's
  to take effect (or anytime in the prior year); late-election relief exists (Rev. Proc. 2013-30).
  Confirm eligibility with `tax-strategy` (≤100 US-person holders, one class).
- **Entity classification — Form 8832:** for an LLC electing C-corp (or to set the default).

## 6. Founder equity (do this early!)
- **Stock purchase / unit issuance agreement** — founders buy shares at a low price while value is
  low (basis + QSBS clock start).
- **§83(b) election — file within 30 days** of a restricted-stock grant (non-negotiable deadline;
  miss it and vesting becomes ordinary income on each tranche). See `equity-and-securities`.
- **IP assignment + confidentiality (CIIAA)** — every founder/employee assigns IP to the company;
  investors require this in diligence.
- **Vesting** (typical 4-year / 1-year cliff) via the stock agreement.

## 7. After formation
- Open the **bank account** (EIN + charter + org consent).
- **Foreign-qualify** in every state with nexus (employees/property/sales).
- Start the **minute book** (charter, bylaws/OA, consents, cap table, stock ledger).
- Calendar the **compliance obligations** (ongoing-compliance-calendar).
