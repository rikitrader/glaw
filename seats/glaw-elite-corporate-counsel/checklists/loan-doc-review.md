# Checklist — Loan / Note / Financing Review

## Parties & authority
- [ ] Correct legal names, entity types, jurisdictions for every party
- [ ] LLC treated as LLC (not "Corporation"/"Bylaws"/"Directors"/"shares")
- [ ] Signer capacity correct; **individual guaranties signed individually**
- [ ] Authorizing resolutions / written consents exist AND are attached
- [ ] Good-standing certificates exist AND are attached
- [ ] **Lender/plaintiff actually signed** the master agreement (no blank block)

## Economics
- [ ] Principal = cash actually advanced? (or does it bake in fees/MCA factor?)
- [ ] Stated rate + default rate + late/per-diem + fees aggregated → effective rate
- [ ] Usury check for governing-law state AND borrower's state
- [ ] Payment ledger exists; face amount reconciled to claimed balance
- [ ] Usury savings clause present?

## Document integrity
- [ ] All schedules/exhibits populated (indebtedness, liens, ownership, debt schedule)
- [ ] Effective date vs. signature vs. certificate vs. notary dates reconciled
- [ ] No "Amendment" recital on a stand-alone agreement (template-reuse signal)
- [ ] Defined terms all defined; cross-references resolve
- [ ] Compare against any other copy of the same instrument clause-by-clause

## Security
- [ ] Security agreement exists and is signed
- [ ] UCC-1 filed in correct jurisdiction; prior UCC-3 terminations obtained
- [ ] Lien search performed; priority confirmed

## Origin & relationships
- [ ] Is this a restructured MCA / defaulted advance relabeled a "loan"?
- [ ] Undisclosed prior notes/debts vs. "no other indebtedness" reps
- [ ] Cross-guarantees by many one-person-controlled affiliates (collusion/FUFTA)
- [ ] Mixed governing law across related instruments
