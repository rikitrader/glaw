# Checklist — Attacking (or Defending) a Judgment

## Posture
- [ ] Party or non-party? (drives Rule 1.540 vs. independent action vs. FUFTA)
- [ ] Within **1 year** (1.540(b)(1)-(3)) or relying on **no-limit** fraud-on-court / void?
- [ ] Appellate deadline (30 days) preserved?

## Collusion / fraud-on-court badges (cite the record for each)
- [ ] Same person/entity controls plaintiff AND defendant sides
- [ ] "**Agreed**"/consent or rapid default judgment
- [ ] Waivers of service; no answer/discovery/contest; weeks to judgment
- [ ] "Service"/notice to the controlling insider's own email
- [ ] Judgment amount on bare face number; no ledger/affidavit
- [ ] **Lost-note** count while a full executed copy exists / original likely held
- [ ] Express lien / in rem preservation (priming other creditors)
- [ ] True nature of debt concealed from court (e.g., defaulted MCA → "loan")

## §673.3091 lost-note stress test
- [ ] Was proponent entitled to enforce when "loss" occurred?
- [ ] Loss NOT by transfer/seizure?
- [ ] Can it really not obtain the original? (does it hold a closing binder?)
- [ ] Was adequate protection actually ordered for the obligor?

## FUFTA overlay (non-party creditor)
- [ ] Transfer/obligation + transferee/beneficiary identified
- [ ] No reasonably equivalent value / actual-intent badges
- [ ] Debtor insolvent or undercapitalized
- [ ] Within 4-yr / 1-yr-discovery window

## Defense neutralizers (have an answer to each)
- [ ] Standing / stranger to the judgment
- [ ] Statutory lost-note remedy is routine; copy properly attached
- [ ] Consent judgments favored; no false statement of fact to court
- [ ] One-year bar / laches / res judicata
