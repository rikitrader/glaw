# Board Approval Checklist

Use this appendix before grants rely on the draft 409A output.

| Item | Status | Evidence |
|---|---|---|
| Board packet includes signed or pending independent-appraiser report. | OPEN | |
| Valuation date is on or before grant approval date. | OPEN | |
| Board consent adopts common FMV and strike price explicitly. | OPEN | |
| Grant list ties recipient, share count, ISO/NSO status, and strike price. | OPEN | |
| Material-event review covers the period through approval. | OPEN | |
| Minutes/consent state no grants are effective before appraiser sign-off. | OPEN | |

Required outcome: `BOARD READY`, `BOARD READY WITH CONDITIONS`, or `BOARD BLOCKED`.
