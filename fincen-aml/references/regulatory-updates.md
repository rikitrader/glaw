# AML Investigator — Regulatory-Change Slice (2025–2026)

The AML-program / CDD / framework slice of the FinCEN Updates corpus. Full ledger and
primary-authority map: **`../fincen/references/regulatory-updates-2025-2026.md`** and
`../fincen/references/bsa-aml-framework.md`. Verify all proposed-rule status on FinCEN.gov.

## Program reform — the big one
- **2026-04-07 — NPRM: Fundamentally Reform FI AML/CFT Programs** `NPRM`. Risk-based,
  **"effective and reasonably designed"** programs implementing **AMLA-2020**; institutions
  must incorporate the national **Priorities**. **Fully supersedes/withdraws the July-2024
  program NPRM.** Authority: 31 U.S.C. 5318(h); AMLA §§ 6101, 6002; 31 CFR 1020.210 et al.
  - *Impact:* re-anchor every covered client's written program to a **documented risk
    assessment** mapped to the Priorities — but **do not rebuild to proposed text as if final.**

## CDD / beneficial ownership (the fifth pillar)
- **2026-02-13 — Exceptive Relief to Streamline CDD** `Exceptive Relief` — reduces certain CDD
  steps; scope set by the order. Authority: 31 U.S.C. 5318(a); 31 CFR 1010.230.
- **2026-05-06 — Consolidated CDD FAQs** aligned to the relief order `FAQ` — now the current
  interpretive baseline. Detail: `../fincen/references/cdd-beneficial-ownership.md`.
- **CTA/BOI reporting** (31 U.S.C. 5336; 31 CFR 1010.380) is **separate** and **in flux** —
  verify current scope before advising any entity.

## Program housekeeping
- **2026-04-30 — RMSB registration OMB renewal (Form 107)** `OMB/PRA` — confirm MSB clients'
  registration is current (31 CFR 1022.380).
- **2026-02-24 — BSAAG nominations** `Admin`; **2026-04-16 — FY2025 Year in Review** `Report` —
  read for exam-emphasis signals.

## Enforcement that teaches AML-program lessons
- **2026-03-06 — Canaccord Genuity $80M** and **2025-12-09 — Paxful $3.5M**: both trace to a
  **deficient AML program + missed SARs**. **2024-01-31 — Asre $100k**: **individual** liability
  for a responsible person. Full lessons: `../fincen/references/enforcement-actions.md`.

## Investigator takeaways
1. The **program is the product** — most enforcement is program-failure + unfiled SARs.
2. Build the **beneficial-ownership tree** before trusting any entity's stated purpose
   (the control + 25%-ownership prongs are the investigative spine).
3. The AMLA shift means "papered but ineffective" is now itself a finding.
