# C. FEDERAL-DOOR GATES (confirm BEFORE drafting — these are jurisdictional/condition-precedent)
1. **Jurisdiction:** a real §1331 hook or complete diversity + > $75,000 (§1332); **Article III standing**
   (a concrete injury — TransUnion).
2. **Exhaustion / presentment:** EEOC charge + right-to-sue letter (Title VII/ADEA — 90-day window);
   FTCA §2675 administrative claim + 6-month wait; ERISA plan remedies; PLRA.
3. **Limitations:** the claim-specific period (securities §13/§1658(b); §1983 borrowed state PI; FCA
   6/10-yr; FTCA 2-yr admin + 6-mo suit) — verify accrual + tolling.
4. **Special procedure:** FCA **under seal** + §3730(b)(2) DOJ disclosure; PSLRA lead-plaintiff +
   discovery stay; class action Rule 23; arbitration agreement (FAA).
5. Venue (§1391) + proper service (Rule 4); cites verified (/glaw-legal-research); federal adversarial
   pass (Federal Defense Counsel / DOJ-AUSA via the gate); UPL/work-product footer.
